## 📋 服务管理

### 查看运行状态
```bash
# 查看所有服务
ps aux | grep -E "(uvicorn|python3.*main.py|node|vite)"

# 查看端口占用
netstat -tuln | grep -E "(8000|9000|3306)"
```

### 重启服务
```bash
# 重启后端API
pkill -f uvicorn
cd backend && nohup uvicorn main:app --host 0.0.0.0 --port 8000 --reload > logs/api_backend.log 2>&1 &

# 重启前端 (在fronted目录下)
cd fronted && npm run dev
```

### 查看日志
```bash
# 后端日志
```
