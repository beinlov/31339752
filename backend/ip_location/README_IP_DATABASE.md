# IP 地理位置数据库配置说明

## 缺失文件
系统需要 `IP_city_single_WGS84.awdb` 文件用于IP地理位置查询。

## 获取方式
1. **从项目团队获取**: 联系项目维护者获取正版数据库文件
2. **使用免费替代方案**: 可以使用 GeoLite2 数据库
3. **临时跳过**: 修改代码暂时禁用IP地理位置功能

## 安装步骤
1. 获取 `IP_city_single_WGS84.awdb` 文件
2. 将文件放置到 `/home/spider/31339752/backend/ip_location/` 目录
3. 确保文件权限正确: `chmod 644 IP_city_single_WGS84.awdb`

## 临时解决方案
如果暂时无法获取数据库文件，可以修改相关代码跳过IP地理位置功能，系统其他功能仍可正常运行。
