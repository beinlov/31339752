# Import Script Modification

## Changes Made

### 1. Added `area` Field to Database

**Script**: `add_area_field.py`

Added new column to `botnet_nodes_utg_q_008` table:
```sql
ALTER TABLE botnet_nodes_utg_q_008 
ADD COLUMN area VARCHAR(50) NULL AFTER city
```

**Table Structure**:
```
country  -> 国家
province -> 省份
city     -> 市
area     -> 区县 (NEW!)
unit     -> 单位
industry -> 行业
```

### 2. Modified Import Logic

**File**: `import_db.py`

#### Before (Merged city and city1)
```python
elif field == 'city':
    c1 = row.get('city', '')
    c2 = row.get('city1', '')
    if c1 and c2:
        val = f"{c1} {c2}"  # Merged: "成都市 朝阳区"
    else:
        val = c1 or c2 or None
```

#### After (Separate fields)
```python
elif field == 'city':
    val = row.get('city', '')  # 市 -> city
    if val == '*' or pd.isna(val):
        val = None

elif field == 'area':
    val = row.get('city1', '')  # 区县 -> area
    if val == '*' or pd.isna(val):
        val = None
```

## Special Handling

### Asterisk (*) Values
- Excel: `*` → Database: `NULL`
- Excel: empty → Database: `NULL`

### Column Name Matching
The script intelligently matches column names:
- `市`, `city`, `shi` → `city` field
- `区县`, `city1`, `qu`, `xian` → `area` field

## Benefits

### 1. Better Data Structure
- ? Separate city and area for better querying
- ? More flexible for filtering by district/county
- ? Cleaner data organization

### 2. Query Examples

**Find all nodes in specific area**:
```sql
SELECT * FROM botnet_nodes_utg_q_008 
WHERE city = '成都市' AND area = '朝阳区';
```

**Count by city and area**:
```sql
SELECT city, area, COUNT(*) as count
FROM botnet_nodes_utg_q_008
GROUP BY city, area
ORDER BY count DESC;
```

**Display full location**:
```sql
SELECT 
    CONCAT(province, ' ', city, ' ', IFNULL(area, '')) as full_location,
    COUNT(*) as count
FROM botnet_nodes_utg_q_008
GROUP BY full_location;
```

## Notes

- `area` field is nullable (can be NULL)
- `*` values are converted to NULL
- Empty strings are converted to NULL
- Field position: after `city` field
