
```bash

```

### 查看提交历史

#### 查看当前分支的提交历史
```bash

```

#### 查看所有分支的提交历史（图形化）


## 分支合并（未来需要时）

### 将 huizhou 合并到 master（需要时）

```bash
# 1. 切换到 master 分支
git checkout master

# 2. 拉取最新的 master
git pull origin master

# 3. 合并 huizhou 分支
git merge huizhou

# 4. 解决冲突（如果有）
# 编辑冲突文件，然后：
git add <冲突文件>
git commit

# 5. 推送到远程 master
git push origin master
```

### 创建 Pull Request（推荐方式）

1. 访问 GitHub 仓库页面
2. 点击 "Pull requests" 标签
3. 点击 "New pull request"
4. 选择：base: `master` ← compare: `huizhou`
5. 填写 PR 描述
6. 创建并等待审核
7. 审核通过后合并
